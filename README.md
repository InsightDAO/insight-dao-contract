# Insight DAO Contract

## Running test 
```shell
npx hardhat test
```
Note: Github action will run the test with the command below for every push. See .github/workflows/github-actions.yml for the config.

## Try other commands

$ npx hardhat

Usage: hardhat [GLOBAL OPTIONS] <TASK> [TASK OPTIONS]
